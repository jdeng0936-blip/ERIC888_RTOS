## 2. 核心板 A（计算板）硬件清单

### 2.1 MCU: STM32F429IGT6
- 180MHz Cortex-M4 + FPU + DSP
- 1MB Flash, 256KB SRAM
- FMC、LTDC、SPI×4、UART×8、ADC×3

### 2.2 外部存储
| 器件 | 型号 | 容量 | 接口 | 备注 |
|------|------|------|------|------|
| SDRAM | W9825G6KH-6 | 256Mb (32MB) | FMC SDRAM | 运行时数据 + LCD 帧缓冲 |
| SPI Flash | W25Q128 | 128Mb (16MB) | SPI3 (CS=PD6) | 固件/配置/故障录波 |
| NAND Flash | W29N01GV | 1Gb (128MB) | FMC NAND | 大容量数据存储 |

### 2.3 AD7606 高速ADC
| 参数 | 值 |
|------|------|
| 通道数 | 8 (同步采样) |
| 分辨率 | 16-bit |
| 接口 | FMC 16-bit 并行 (Bank1 NE4) |
| 基地址 | 0x6C000000 |
| CONVST | PC6 (TIM3_CH1 PWM, 25.6kHz) |
| BUSY | PB1 (EXTI1, priority=4) |
| RESET | PG7 |
| CS | PG12 (FMC_NE4) |

### 2.4 模拟前端（背板）
| 器件 | 型号 | 用途 |
|------|------|------|
| 仪表放大器 | AD8226 ×8 (U35~U42) | 差分电压/电流采样 |
| 电流传感器 | ACS725AB-5A (U14,U15,U30,U31) | Hall 效应电流检测 |
| 负电源 | MT3608L (U52) | Boost 生成 -12V (运放供电) |

### 2.5 K-Switch 分合闸驱动（背板）
| 通道 | ON 引脚 | OFF 引脚 | 驱动链 |
|------|---------|----------|--------|
| K1 | PH14 | PH15 | MCU → ACPL247光耦 → QX3120/TLP250 → KNX42120 MOSFET |
| K2 | PI4 | PH13 | 同上 |
| K3 | PI6 | PI5 | 同上 |
| K4 | PI8 | PI7 | 同上 |

- 反馈信号：CF1~CF4（ON/OFF 位置确认）
- MOSFET：KNX42120（大功率型）

### 2.6 通信接口（A板侧）

| 外设 | 引脚 | 用途 |
|------|------|------|
| **SPI4** | PE2(SCK), PE5(MISO), PE6(MOSI), PE4(NSS) | 板间通信（RS422 隔离） |
| **PI11** | 中断 | SPI4 IRQ 握手信号 |
| **UART3** | PB10(TX), PB11(RX), PB8(TXEN) | RS485 通道 |
| **USART6** | — | 4G 模块（A板侧备用） |
| **UART7** | — | GPS（A板侧备用） |

### 2.7 内部 ADC（STM32 ADC1）
- 6 通道 DMA 连续采样（温度、参考电压等）
- TIM3 触发

### 2.8 数字IO（背板）
| 信号 | 引脚 | 类型 |
|------|------|------|
| ESS 输入 | PG11~PG14 | ACPL247 隔离 DI |
| 位置反馈 | PH13~PH15, PI1~PI3, PA10, PA12 | ON/OFF 状态 |
| 继电器输出 | — | 5路 SH1-05DC-SL-2C (JX1~JX5) |

## 3. 核心板 B（面板板）硬件清单

### 3.1 MCU: STM32F429IGT6（第二片）
- 与 A 板完全相同的 MCU

### 3.2 LCD 显示

| 参数 | 值 |
|------|------|
| 接口 | **LTDC RGB 24-bit 并行** |
| 分辨率 | 800×480 或 1024×600 |
| 偏压生成 | TPS65150 |
| 背光驱动 | PT4110E89E (恒流) |
| 背光PWM | **PA11** (TIM1_CH4) |
| LCD复位 | **PA7** (LCDRST) |

**RGB 引脚映射：**

| 颜色 | 位 | 引脚 |
|------|-----|------|
| R0~R7 | 8-bit | PI15, PJ0, PJ1, PJ2, PJ3, PJ4, PJ5, PJ6 |
| G0~G7 | 8-bit | PJ7, PJ8, PJ9, PJ10, PJ11, PK0, PK1, PK2 |
| B0~B7 | 8-bit | PJ12, PJ13, PJ14, PJ15, PK3, PK4, PK5, PK6 |

**LTDC 控制线：**

| 信号 | 引脚 |
|------|------|
| HSYNC | PI12 |
| VSYNC | PI13 |
| CLK | PI14 |
| DE | PK7 |

## 4. 电源架构

```
AC220V ──► 14D561K 压敏电阻保护
       ──► POW1 (AC-DC) → 15V（MOSFET 栅极驱动）
       ──► POW2 (AC-DC) → 12V（模拟/开关主电轨）
              │
              └──► NS6326B Buck → 5V（数字主电轨）
                      │
                      ├──► PW6599B LDO → 3.3V-1（模拟隔离域）
                      ├──► PW6599B LDO → 3.3V-3（数字IO域）
                      └──► BP6599 LDO → 3.3V（B板数字域）

              └──► MT3608L Boost → -12V（运放负电源）

隔离供电：B0505S-1WR2 DC-DC → 各 RS485/RS422 通道独立 5V 域
```

## 5. 关键差异：A板 vs B板

| 特性 | A板 (Calc) | B板 (Panel) |
|------|-----------|-------------|
| 主要职责 | 高速采样 + DSP + 保护跳闸 | 显示 + 通信 + 存储 |
| AD7606 | ✅ FMC NE4 直连 | ❌ |
| K-Switch | ✅ GPIO → 光耦 → MOSFET | ❌ |
| LCD | ❌ | ✅ RGB LTDC 800×480 |
| W5500 | ❌ | ✅ SPI2 |
| 4G | ❌ (USART6 备用) | ✅ UART7 (EC20) |
| GPS | ❌ (UART7 备用) | ✅ UART1 |
| SD Card | ❌ | ✅ SDIO |
| RS485 | ✅ UART3 (1路) | ✅ UART4+UART2 (2路) |
| SPI Flash | ✅ SPI3 (W25Q128) | ❌ |
| SDRAM | ✅ W9825G6KH | ✅ W9825G6KH |
| 板间SPI4 | Slave | Master |
| 以太网 PHY | DP83848 (RMII) | ❌ (用 W5500) |

## 6. P2 开发所需关键信息汇总

### B板 LCD 驱动开发
- 控制器：**STM32F429 内置 LTDC**（不是外部 LCD 控制器）
- 需要配置 LTDC 时序参数（HSYNC/VSYNC/DE/CLK）
- 背光：PA11 TIM1_CH4 PWM
- 推荐框架：**LVGL** + LTDC + DMA2D

### B板 触摸屏驱动开发
- 控制器：**STMPE811QTR**（I2C 地址通常 0x41）
- I2C1：PB8(SCL) + PB9(SDA)
- 中断：PH7
- ST 提供 HAL BSP 驱动

### B板 W5500 驱动开发
- SPI2：PB10(SCK), PC2(MISO), PC3(MOSI), PB12(CS)
- INT：PF11，RST：PC0
- 推荐库：ioLibrary_Driver (WIZnet 官方)

### B板 4G 模块驱动开发
- EC20 通过 UART7 (PF7/PF6) AT 指令控制
- 电源控制：PF8 高电平开机
- 需要 AT 指令解析器

### B板 GPS 驱动开发
- UART1 (PA9/PA10) 接收 NMEA 语句
- 需要 NMEA 0183 解析器
